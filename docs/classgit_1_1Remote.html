git::Remote Class Reference

Detailed Description

An abstraction of a git remote repository (such as "origin").

A remote has a name and a URL which can be accessed via get_name() and get_url(), respectively. In addition, a non-owning pointer to the underlying git_remote object can be retrieved via get().
