git::Error Class Reference

Detailed Description

An exception class carrying an error code and message.

Error is used as the standard exception class throughout libgit4cpp. It can be used directly or inherited from.

try
{
throw git::Error(errorcode, "An error has occurred");
// or throw git::Error("Another error occurred");
}
catch (const git::Error& e)
{
std::cerr << e.what() << "\n";
}
