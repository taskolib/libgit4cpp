◆ get_dummy_credentials_callback()

CredentialsCallback git::get_dummy_credentials_callback ( )

Return a dummy credentials callback function for libgit2.

The returned function pointer may be called by libgit2 when it needs credentials to access a remote, e.g. over an HTTPS connection. Currently, this is just a dummy implementation that always returns the same username and password.

Referenced by git::Remote::list_references(), and git::Repository::push().
